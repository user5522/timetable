## features

| Feature                                              | Availability    |
|------------------------------------------------------|-----------------|
| Grid view                                            |        ✔        |
| Day view                                             |        ✔        |
| CRUD subjects in both views                          |        ✔        |
| Settings                                             |        ✔        |
| Rotation weeks                                       |        ✔        |
| Edit timetable time period                           |        ✔        |
| Material You                                         |        ✔        |
| Dark, Light, System themes                           |        ✔        |
| Current day indicator (Grid)                         |        ✔        |
| Multiple timetables                                  |        ✔        |
| Hide bottom navbar <br>(more screen real estate)     |        ✔        |
| Localization                                         | English, French |
